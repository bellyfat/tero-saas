Tero

A Home Surveillance Software implemented in Python

Acerca de

Tero es un software hecho en el lenguaje de programación Python, que te permite saber si alguien entra a tu casa cuando no estas. Para lograr esto, Tero precisa de:

  • Camaras IP que soporten detección de movimientos y subir fotos por FTP. Cualquier camara que soporte estas dos cosas deberia funcionar.
  • Que tengas una cuenta de Telegram en tu smartphone
  • Una maquina donde instalar y dejar corriendo a Tero (por ej: un VPS de 5 USD de Digital Ocean funciona genial!)

Instalar

  1. Crea un virtualenv (debes tener instalado virtualenvwrapper) para instalar Tero con el comando:

    $ mkvirtualenv tero

  2. Teniendo el virtualenv activo, instala las dependencias:

    $ pip install -r requirements.txt

  3. Crea la base de datos, tablas, etc. Por default se utiliza sqlite:

    $ ./manage.py migrate

  4. Corre el servidor FTPd (donde las cámaras IP enviaran las imágenes) con el comando:

    $ ./manage.py run_ftpd

  5. (Opcional) - Si deseas configurar los usuarios utilizando el admin de Django, corre el comando:

    $ ./manage.py runserver

Como funciona

  1. Instalas Tero y arrancas el servidor FTPd (el admin de django es opcional).
  2. Si no tienes una cuenta de Telegram, tienes que crear una. Al momento Tero solo sabe notificarte mediante Telegram. También activaras y desactivaras la alarma utilizando Telegram.
  3. Agrega/configura los usuarios de tu alarma Tero mediante la linea de comando o utilizando el admin de Django. Recuerda que puedes asociar una misma alarma a varios usuarios, esto posibilita por ejemplo que toda tu familia pueda activar/desactivar y recibir alertas de la alarma de tu casa (relacionas diferentes UserProfile al mismo objeto AlarmProfile).

Tero vigila tu casa en tres etapas:

  • Las cámaras IP cuando detectan movimiento envían las imágenes al servidor FTP de Tero.
  • Si la alarma se encuentra activa, y Tero recibe una imagen desde las cámaras, enviara la imagen por Telegram a las personas (UserProfile) que hayas relacionado con el AlarmProfile. Si la alarma se encuentra desactivada, Tero borrara la imagen enviada por la cámara y no envía ninguna notificación.
  • El bot de Telegram que has creado/configurado en Tero, te permite:
    • Activar la alarma escribiendo: /ac o bien /activate
    • Desactivar la alarma escribiendo: /de o bien /deactivate
    • Conocer el status de la alarma con el comando: /st o bien /status

Estado del proyecto

Actualmente estoy utilizando Tero en casa, y me da mas tranquilidad tenerlo funcionando que no tener nada. Igualmente Tero es un prototipo, le faltan MUCHAS cosas, por lo cual tu contribución es bienvenida!

Algunas features que me gustaría Tero tenga próximamente son:
  • Haga sonar una alarma/buzzer. Si puede hacer el sonido de 'tero tero!' aun mejor!
  • Procesamiento de imagenes para descartar falsos positivos (hay veces que se cruzan bichitos y hacen trigger de la alarma jeje)
  • Una versión que corra en RPi/C.H.I.P/Omega2, mas que nada para activar/desactivar el buzzer
  • Un API Rest para interactuar con Tero

Configurando usuarios y alarmas

Tero provee algunos comandos que pueden ser ejecutados desde la consola.

  1. Para agregar usuarios utiliza:

    $ ./manage.py create_user --help

  2. Para borrar un usuario utiliza:

    $ ./manage.py rm_user --help
