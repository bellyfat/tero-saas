Utilizando Tero con camaras TPLink nc220

En casa estoy utilizando camaras TPLink NC220 las cuales me parecieron una buena opción por:

  1. Visión nocturna hasta 5 mts
  2. Detectan movimiento y se las puede configurar para que suban las imágenes que captan por FTP
  3. Detectan sonido y pueden ejecutar un alerta ante este evento
  4. Tiene un app mobile con el cual puedes acceder a las cámaras en cualquier momento desde tu smartphone
  5. La relación precio/prestaciones me pareció excelente.
  6. Bonus Point: Trae un modo 'wireless extender', esto quiere decir que puedes configurar la cámara para que 'extienda' la cobertura de tu red wifi en tu casa, genial!

Al día de hoy esta cámara puede conseguirse por 900$ Pesos Argentinos aprox en: http://articulo.mercadolibre.com.ar/MLA-616702885-camara-ip-tp-link-nc220-cloud-wi-fi-300mbps-diurna-nocturna-_JM

Configuración y Deploy

  1. Se hace mediante el navegador web.
  2. La parte de 'motion detection' hay que hacerlo si o si desde una maquina Windows con Firefox (punto para abajo)

  3. Luego se configura 'Notification Delivery', y se selecciona FTP con los valores:

    • FTP Server/Port: IP y puerto donde esta corriendo el server FTP de Tero
    • Username: Usuario configurado en Tero
    • Password: Password del usuario en Tero
    • Path: Lo dejamos como viene, en '/'
    • Cliqueamos el checkbox 'Passive Mode'

Listo! Cada vez que la cámara detecte movimiento, enviara las imágenes al FTP server de Tero. Luego Tero me envia las imagenes a mi smartphone mediante Telegram.
